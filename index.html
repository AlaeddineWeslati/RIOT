RIOT OS

The Revolutionary IoT Operating System

Download this project as a .zip file Download this project as a tar.gz file

RIOT leverages a microkernel to provide an energy-efficient operating system for IoT solutions.

Developer Friendly

  • Program in C or C++ like you're used to: 
#include <stdio.h>

int main(void) {
    puts("Hello world!\n");
    while(1);
}
  • Standard development tools like gcc, gdb ...
  • Minimized hardware dependent code
  • Zero learning curve embedded programming

Efficient

  • Robustness & code-footprint flexibility due to the microkernel architecture
  • Tickless scheduler enabling maximum energy-efficiency
  • Real-time capability due to ultra-low interrupt latency and priority-based scheduling
  • Multi-threading with ultra-low threading overhead
  • Code once, run both on 16-bit platforms (e.g. MSP430) and on 32-bit platforms (e.g. ARM)

IoT Ready

  • IoT network stack
    • 6LoWPAN
    • IPv6
    • RPL
    • TCP and UDP
  • Flexible memory management (static and dynamic memory allocation)
  • High resolution and long-term timers
  • FAT file system
  • Logging system
  • Wiselib support (C++ algorithm library, including routing, clustering, timesync, localization, security and more algorithms)
