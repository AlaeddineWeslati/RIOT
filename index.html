RIOT OS

RIOT - The revolutionary IoT OS!

Developer friendly

  • program in C or C++ like you're used to

a complete hello world application looks like this:

#include <stdio.h>

int main(void) {
    puts("Hello world!\n");
    while(1);
}
  • standard development tools like gcc, gdb, or ...
  • minimized hardware dependent code
  • zero learning curve embedded programming

Performance

  • increased robustness due to the microkernel architecture
  • the tickless scheduler enables maximum energy-efficiency
  • low interrupt latency and priority based scheduling make RIOT real-time capable
  • minimum threading overhead
  • 100% modular structure allows RIOT to run on tiny platforms as well as on more powerful systems

Features

  • IoT network stack
    • 6LoWPAN
    • IPv6
    • RPL
    • TCP and UDP
  • flexible memory managment (static and dynamic memory allocation)
  • high resolution and long-term timers
  • FAT file system
  • logging system
  • Wiselib support (C++ algorithm library, including routing, clustering, timesync, localization, security and more algorithms)
